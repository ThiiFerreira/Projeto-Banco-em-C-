## Como executar: 
Baixar ou clonar o repositorio e execute o arquivo ProjetoBanco.exe na pasta ProjetoBanco\bin\Debug, quando abrir o executavel vai pedir pra fazer login, os dados do login estÃ£o na mesma pasta debug no arquivo contas.txt, pode escolher qualquer uma e testar as funcionalidades.
